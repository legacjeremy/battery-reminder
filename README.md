# BattTrack

BattTrack est une PWA pour suivre les batteries rarement utilisées, estimer leur autodécharge et éviter les décharges profondes.

## Version

v1.3.1

## Changements v1.3.1

- Correction du remplissage des icônes batterie.
- Blocage du scroll arrière-plan quand une modale est ouverte.
- Ajout de `version.json`.
- Ajout de `CHANGELOG.md`.
- Affichage des nouveautés avant mise à jour.
- Rappel de sauvegarde avec suivi du dernier export.
- Recherche dans le nom et les notes.
- Mini graphique simple dans la fiche batterie.
